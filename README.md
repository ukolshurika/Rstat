Rstat
=====

Домашние задания по пакету R
1-3 в разработке.

5 задание принято А.И. Мартыненко. Можно подставлять свои данные, но ввод там не оформлен!

## Запуск
```bash
./start_R.sh <файл с программой> <номер варианта>
```
Например:
```bash
./start_R.sh 1.R 10
```
